# 快速入门

> 创建你的第一个 Agent Skill，并在 VS Code 中看到它运行。

本教程要做一个技能：**让 AI 能掷骰子**（用随机数生成器）。

---

## 前置条件

- **VS Code + GitHub Copilot**

> 💡 提示：教程用 VS Code 演示，但 Agent Skills 是开放格式。同样的技能在 Claude Code、OpenAI Codex 等任何兼容的 Agent 上都能用。

---

## 创建技能

### 文件位置

```
.agents/skills/roll-dice/SKILL.md
```

> VS Code 默认在 `.agents/skills/` 目录下查找技能。

### SKILL.md 完整内容

```markdown
---
name: roll-dice
description: Roll dice using a random number generator. Use when asked to roll a die
---

To roll a die, use the following command that generates a random number from 1 to the given number of sides:

```bash
echo $((RANDOM % <sides> + 1))
```

**powershell**
Get-Random -Minimum 1 -Maximum (<sides> + 1)
```

Replace `<sides>` with the number of sides on the die (e.g., 6 for a standard die, 20 for a d20).
```

### 三个部分的作用

| 部分 | 说明 |
|------|------|
| **name** | 技能短标识符，必须和文件夹名一致 |
| **description** | 告诉 Agent 何时激活这个技能（匹配关键词） |
| **body（正文）** | Agent 激活后执行的指令（这里教它怎么生成随机数） |

> 就一个文件，不到 20 行！

---

## 试一试

1. 用 VS Code 打开你的项目
2. 打开 Copilot Chat 面板
3. 在底部模式选择框中选择 **Agent mode**
4. 输入 `/skills` 确认 `roll-dice` 出现在列表中（如果没有，检查 `.agents/skills/roll-dice/SKILL.md` 路径）
5. 对它说：**"Roll a d20"**

Agent 应该会：
- 自动激活 `roll-dice` 技能
- 可能会请求允许执行终端命令 → 点允许
- 运行命令并返回 1-20 之间的随机数

> ⚠️ **工具调用可靠性因模型而异** —— 有些模型会严格遵循指令并运行命令，有些可能尝试自己回答。如果 Agent 没有运行命令，试试从模型下拉菜单换个模型。

---

## 工作原理

整个流程分三步：

| 步骤 | 说明 |
|------|------|
| **1. 匹配** | 你问"掷骰子"，Agent 把你的问题和技能描述匹配，发现 `roll-dice` 技能相关 |
| **2. 激活（Activation）** | Agent 加载完整的 `SKILL.md` 正文到上下文中 |
| **3. 执行（Execution）** | Agent 按照指令操作，把终端命令中的面数替换成你请求的数字 |

这个过程用了 **渐进式披露（progressive disclosure）** —— 让 Agent 可以访问很多技能，而不需要一开始就加载所有指令。
