# Kotlin 变量与数据类型

> 📖 来源：飞书文档 / Kotlin 学习路线

---

## 1. val 与 var —— 不可变变量与可变变量

```kotlin
// val: 不可变（类似 Java 的 final），赋值后不能再改
val name = "Kotlin"
// name = "Java"  // 编译错误！val 不能重新赋值

// var: 可变，可以重新赋值
var age = 25
age = 26  // OK
```

**建议**：优先使用 `val`，除非确实需要修改。

## 2. 基本数据类型 —— Byte, Short, Int, Long, Float, Double, Boolean, Char

```kotlin
// 整数类型
val byte: Byte = 127          // 8位，范围 -128 ~ 127
val short: Short = 32767      // 16位
val int: Int = 2147483647     // 32位，最常用
val long: Long = 9999999999L  // 64位，注意末尾的 L

// 浮点类型
val float: Float = 3.14f      // 32位，注意末尾的 f
val double: Double = 3.141592653589793  // 64位，默认小数就是 Double

// 布尔类型
val isKotlinFun: Boolean = true

// 字符类型
val letter: Char = 'A'
```

## 3. 类型推断 —— 编译器自动推断变量类型

```kotlin
// Kotlin 编译器很聪明，大多数时候不需要显式写类型

val city = "Beijing"       // 推断为 String
val score = 98             // 推断为 Int
val pi = 3.14              // 推断为 Double
val passed = true          // 推断为 Boolean

// 可以用 ::class 查看类型（了解即可）
println("city 的类型: ${city::class.simpleName}")    // String
println("score 的类型: ${score::class.simpleName}")  // Int
println("pi 的类型: ${pi::class.simpleName}")        // Double
```

## 4. 字符串模板 —— $变量 和 ${表达式}，多行字符串

```kotlin
val language = "Kotlin"
val version = 2.0

// 简单变量引用：$变量名
println("我在学习 $language")

// 表达式引用：${表达式}
println("$language $version 发布于 ${2024 - 1}年... 开玩笑的，是2024年")

// 多行字符串（trimIndent 去掉前导空格）
val poem = """
    白日依山尽，
    黄河入海流。
    欲穷千里目，
    更上一层楼。
""".trimIndent()
println(poem)
```

## 5. 类型转换 —— 显式转换 toXxx()，安全转换 toIntOrNull()

```kotlin
// Kotlin 不会隐式转换类型，需要显式调用 toXxx()

val intNum: Int = 42
val longNum: Long = intNum.toLong()     // Int -> Long
val doubleNum: Double = intNum.toDouble() // Int -> Double
val strNum: String = intNum.toString()  // Int -> String
println("Int=$intNum, Long=$longNum, Double=$doubleNum, String=\"$strNum\"")

// 字符串转数字
val str = "123"
val parsed: Int = str.toInt()
println("字符串 \"$str\" 转为 Int: $parsed")

// 安全转换（转换失败返回 null，不会崩溃）
val badStr = "abc"
val safeResult: Int? = badStr.toIntOrNull()
println("\"$badStr\" 转 Int 结果: $safeResult")  // null
```

## 6. 编译期常量 —— const val

```kotlin
// const val 只能用在顶层或 object 中（不能在函数里）
// 编译期常量，必须是基本类型或 String
const val MAX_COUNT = 100
```

## 7. 数组 —— intArrayOf / arrayOf 的基本用法

```kotlin
val numbers = intArrayOf(1, 2, 3, 4, 5)
val names = arrayOf("Alice", "Bob", "Charlie")

println("数组第一个元素: ${numbers[0]}")
println("名字列表: ${names.joinToString()}")
println("数组大小: ${numbers.size}")
```
