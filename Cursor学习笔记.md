# Cursor 学习笔记

> 记录 Cursor 编辑器的使用技巧、快捷键、AI 功能与工作流。

---

## 一、基础概念

- **Cursor**：基于 VS Code 的 AI 编程编辑器
- **核心能力**：AI 补全、对话、重构、解释代码等

---

## 二、常用功能

### 2.1 AI 对话 (Chat)
- 快捷键：`Ctrl + I`
- 用途：提问、写代码、改 bug、解释代码

### 2.2 代码补全 (Tab)
- 输入时自动出现 AI 建议，按 `Tab` 接受

### 2.3 选中代码后操作
- 选中代码 → 右键或快捷键 → 可让 AI 解释、重构、写测试等

### 2.4 打开终端 (Terminal)
- 快捷键：`Ctrl + J`
- 用途：显示/隐藏底部面板（含集成终端），方便运行命令、脚本

---

## 三、技巧与工作流

- 用 `@` 引用文件、文件夹、文档（如 `@文件名`、`@docs`）
- 在对话中说明项目技术栈和约定，提高生成质量
- 复杂需求用 Composer/Agent，可跨文件修改

---

## 四、快捷键速查

| 功能         | 快捷键    |
|--------------|------------|
| 打开 AI 对话 | Ctrl + I   |
| 打开/关闭终端 | Ctrl + J   |
| 接受补全     | Tab        |
