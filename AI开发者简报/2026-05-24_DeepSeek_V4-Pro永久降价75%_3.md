# DeepSeek V4-Pro 永久降价 75%：国产模型的极致性价比战略

## 一句话总结
DeepSeek V4-Pro API 永久降价 75%，输入 0.435 美元/百万 token，比 GPT-5.5 便宜约 98%，全部运行在华为昇腾芯片上，同时组建 Harness 团队对标 Claude Code，性价比战略从价格延伸到产品能力。

## 适用场景
- 正在评估大模型 API 成本的开发者
- 使用 DeepSeek API 的团队
- 关注国产 AI 芯片生态的工程师

## 详细内容

| 模型 | 输入价（$/百万 token） | 输出价（$/百万 token） | 缓存命中 |

### 为什么敢永久降价

1. **芯片成本优势**：V4-Pro 全部运行在华为昇腾芯片上，降价底气来自芯片成本结构与英伟达路线的差异。这意味着 DeepSeek 不需要承担英伟达 GPU 的高溢价。

2. **MoE 架构效率**：1.6 万亿总参数但每次只激活 490 亿，推理成本远低于同等参数量的密集模型。

3. **战略意图**：永久降价（而非限时优惠）是明确的市场份额抢占策略。通过将价格锁定在极低水平，DeepSeek 试图将竞争对手挤出性价比敏感的市场。

### Harness 团队：对标 Claude Code

DeepSeek 已在内部组建全新的 **Harness 团队**，主攻代码智能体产品，对标 Anthropic 的 Claude Code。这标志着 DeepSeek 的竞争维度从「模型性价比」延伸到「Agent 产品能力」。

### 对开发者的实际建议

- **API 成本优化**：如果你的工作涉及大规模推理（尤其是代码生成、文本处理），DeepSeek V4-Pro 的性价比目前是市场最优。
- **关注昇腾生态**：DeepSeek 在华为昇腾上的成功案例，对国产 AI 芯片生态有积极信号。
- **Harness 团队值得跟踪**：如果 DeepSeek 推出类似 Claude Code 的代码 Agent，可能成为性价比更高的替代方案。

### 注意事项

- 5 月 31 日折扣到期后价格才正式锁定，在此之前仍按原价折扣执行。
- 同周特殊字符触发模型异常回复事件引发关注（模型「幻觉」问题），安全性仍需观察。
- Artificial Analysis 测试中 token 消耗可能高于标价暗示的水平。
